## axios について

- lib/axios.ts に axios インスタンスの作成
```
export const apiClient = axios.create({
  baseURL: process.env.NEXT_PUBLIC_API_URL,
  withCredentials: true, // 毎回 Cookie（アクセス・リフレッシュ・CSRF）を自動送信, CORS で allow_credentials=True とセットで使う
});

- 基本的な流れ（GET / POST）
```
const res = await apiClient.get("/csrf-token");

Cookie を自動で送信

レスポンスを JSON で返す

失敗すると axios は「例外 throw」する

- POST の例
```
apiClient.post("/register", userData, {
  headers: {
    "X-CSRF-Token": csrfToken,
  },
});

userData を JSON にして送信

Cookie（csrf_refresh_token, refresh_token, access_token）を自動送信

FastAPI が CSRF・Cookie を検証

成功したら JSON を返す

失敗したらエラーを throw する（status, detail など取得できる）

- try/catch を使う理由

```
try {
  const res = await apiClient.post("/register", data);
} catch (error) {
  console.log(error.response.status);
}

HTTP エラー（400/403/500など）の場合：

error.response.status

error.response.data.detail

を参照してフロント側の処理が書ける。
